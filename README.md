# BA-Source-Dump

- Dump BA Source Data using Il2CppInspectorRedux.
- Keep track of the latest config/resource/data files.

This workflow runs every weekday at 03:00, 05:00, 07:00, 09:00 UTC.
And runs everyday at 07:00
